FilteringButton

@Composable
fun FilteringButton(isSelected: Boolean, @StringRes text: Int, cornerRadius: Dp, paddingVertical: Dp, onButtonClick: () -> Unit, modifier: Modifier = Modifier)

온보딩 과정에서 필터링을 설정할 때 사용되는 버튼입니다.

버튼 클릭을 클릭했을 때, 버튼의 배경 색과 텍스트 색상이 바뀝니다.
